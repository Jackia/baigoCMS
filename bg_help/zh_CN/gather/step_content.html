内容解析

在编辑采集点节目，点内容解析标签，可以对内容解析作出修改。本系统采用 jQuery 选择器来解析 HTML 元素，关于选择器，你可以点上方的选择器帮助来查看常用的选择器。

主要内容解析、附加内容解析

内容解析的填写方法基本一样。

填写说明

标题解析、内容解析、时间解析、文章内容解析、作者解析

选择器
指包含标题的 HTML 元素的选择器。
采集属性
可选以下 3 类，也可以在右侧选择常用属性。
  1. html：采集 HTML 代码
  2. text：采集纯文本
  3. HTML 属性：采集 HTML 属性（如 src、href、name、data-src 等）
    示例：<a href="http://www.domail.com">链接</a>，红色部分为属性（名称="值"）。
过滤器
过滤器可以过滤掉不需要的元素，多个值请用 | 分隔。
  1. 带 - 符号：移除该标签以及标签内容（此时可以为任意的 jQuery 选择器，如：-p|-.title）。
  2. 无符号：
    • 当 [采集属性] 值为 html 时，表示要过滤掉的 HTML 标签。
    • 当 [采集属性] 值为 text 时，表示要保留的 HTML 标签（此时可以为 HTML 标签，如：div|strong）。
替换
替换器可以将“查找”字符更换为“替换”字符（为空则删除“查找”字符），用法与 Word、WPS 的查找替换类似，点 + 按钮，可以创建多个替换器。

其他

填写说明

保留标签

系统将会剔除文章内容中除系统保留标签以外的所有标签，系统保留标签可以点 ? 按钮查询，请在此输入其他需要保留的标签，多个标签请用 | 分隔。

图片过滤关键词

系统在采集数据时，会自动将图片抓取到本地服务器，而有些图片，比如装饰、按钮、广告等是不需要的，管理员可以在此输入关键词，一旦图片的路径中包含有这些内容，将自动忽略这些图片。

属性过滤

以下参数只针对文章内容，默认将剔除所有属性。此处的属性是指系统保留的标签以及管理员指定保留标签的属性。

允许的属性
这些属性将被全部保留，多个属性请用 | 分隔。
忽略标签
这些标签的属性全部保留，多个标签请用 | 分隔。
例外
这些标签的特定属性将被保留，同一标签下的多个属性请用 | 分隔。点 ? 按钮可以查询系统默认保留的属性。
