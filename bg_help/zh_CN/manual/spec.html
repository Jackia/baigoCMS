专题分发配置

文件名：opt_spec.inc.php

定义专题分发至其他服务器的配置，主要为 FTP 以及认证信息。

配置项说明

常量 说明 示例 说明
BG_SPEC_URL URL 前缀 http://spec.domain.com 专题分发目标服务器的 URL 前缀
BG_SPEC_FTPHOST 分发目标服务器的 FTP 地址 spec.domain.com FTP 认证信息详情请咨询服务器提供商
BG_SPEC_FTPPORT 服务器端口号 21  
BG_SPEC_FTPUSER 用户名    
BG_SPEC_FTPPASS 密码    
BG_SPEC_FTPPATH FTP 远程路径 /web/wwwroot/spec 目标服务器所在的目录，末尾请勿加 /
BG_SPEC_FTPPASV 是否开启被动模式 true true 为开启，false 为关闭。
