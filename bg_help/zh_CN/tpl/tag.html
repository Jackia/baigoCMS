TAG 显示

文件名：tag_show.php

用于显示所有与此 TAG 关联的文章。

键名 类型 说明 备注
$this->tplData['tagRow'] array 当前 TAG 详细信息 查看 $this->tplData['tagRow']。
$this->tplData['articleRows'] array 文章列表 所有与此 TAG 关联的文章。详情请查看 文章。
$this->tplData['pageRow'] array 分页参数 详情请查看 分页参数
 

$this->tplData['tagRow'] 数组

当前 TAG 详细信息

键名 类型 说明 备注
tag_id int TAG ID
tag_name string TAG 名称
tag_article_count string 文章计数 与当前 TAG 关联的文章计数。
tag_status string TAG 状态 pub 为发布，hide 为隐藏。
urlRow array 当前 TAG URL 数组 tag_url 为当前 TAG URL，page_attach 为分页附加参数，主要用于分页，page_ext 扩展名（仅用于纯静态模式）。
rcode string 返回代码 显示当前 TAG 的状态，详情请查看 返回代码。
 

TAG 数据示例

print_r($this->tplData['tagRow']) 

Array (
    [tag_id] => 94 //TAG ID
    [tag_name] => asp //TAG 名称
    [tag_status] => show //状态
    [tag_article_count] => 0 //关联文章数
    [urlRow] => Array (
        [tag_url] => /cms/tag/tag-asp/ //URL
        [page_attach] => page- //分页附加
        [page_ext] => .html
    )
    [rcode] => y130102
