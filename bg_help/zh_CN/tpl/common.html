栏目树

在任何模板内，均可以用遍历 $this->tplData['cateRows'] 的方式来显示网站所有栏目的树形结构，详细字段请查看 栏目。

 

栏目树数据示例

print_r($this->tplData['cateRows']) 

Array (
    [7] => Array (
        [cate_id] => 7 //栏目 ID
        [cate_name] => baigo CMS //名称
        [cate_alias] => baigocms //别名
        [cate_parent_id] => 0 //隶属栏目 ID
        [urlRow] => Array (
            [cate_url] => /cate/baigocms/id-7/ //URL
            [cate_urlMore] => /cate/baigocms/id-7/ //更多分页 URL
            [page_attach] => page- //分页附加
            [page_ext] => .html
        )
        [cate_childs] => Array ( //子栏目
            [13] => Array (
                [cate_id] => 13
                [cate_name] => test
                [cate_alias] => test
                [cate_parent_id] => 7
                [urlRow] => Array (
                    [cate_url] => /cate/baigocms/test/id-13/
                    [cate_urlMore] => /cate/baigocms/id-7/ //更多分页 URL
                    [page_attach] => page-
                    [page_ext] => .html
                )
            )
        )
    )
)

包含

在任何模板内，均可以用 include('包含模板路径') 的方式来包含并执行文件，您可以在模板目录下，建立一个目录，如 inc，用来统一存放被包含的模板，如：include('inc/head.php')。

在任何模板内，均可以用 ubb 函数 fn_ubb($string) 的方式来实现部分 UBB 代码，被支持的 UBB 代码请查看 管理后台。

常量

在任何模板内，均可以用 echo 常量名 的方式来输出系统内置常量，如果您想得到所有的常量名，请查看 ./bg_config 目录下的文件。

常用常量
常量名 说明
BG_URL_ROOT 网站根目录
BG_URL_HELP 帮助信息路径
BG_URL_ARTICLE 文章路径
BG_URL_ATTACH 附件路径
BG_URL_SSO baigo SSO 路径
BG_URL_PUB 前台路径
BG_URL_CONSOLE 管理后台路径
BG_URL_INSTALL 安装程序路径
BG_URL_STATIC 静态文件路径
BG_SITE_NAME 网站名称
BG_SITE_DOMAIN 网站域名
BG_SITE_URL 网站首页 URL
BG_SITE_PERPAGE 每页显示数
BG_SITE_TIMEZONE 所处时区
BG_SITE_DATE 日期格式
BG_SITE_DATESHORT 短日期格式
BG_SITE_TIME 时间格式
BG_SITE_TIMESHORT 短时间格式
BG_VISIT_PAGE 静态页数

自定义字段

在任何模板内，均可以用遍历 $this->tplData['customRows'] 的方式来显示所有有效自定义字段的列表。

自定义字段
键名 类型 说明
custom_id int 自定义字段 ID
custom_name string 自定义字段名称
custom_childs array 子字段数组
 

自定义字段数据示例

print_r($this->tplData['customRows']) 

Array (
    [0] => Array (
        [custom_id] => 4 //字段 ID
        [custom_name] => 尺寸 //名称
        [custom_parent_id] => 0 //隶属字段 ID
        [custom_cate_id] => 0 //隶属栏目 ID
        [custom_childs] => Array ( //子字段
            [0] => Array (
                [custom_id] => 5
                [custom_name] => 长
                [custom_parent_id] => 4
                [custom_cate_id] => 2
            )
        )
    )
)

友情链接

友情链接可以通过调用来显示，调用后返回的数组结构如下。

友情链接
键名 类型 说明
link_id int 链接 ID
link_name string 链接名称
link_url string 链接地址
link_blank bool 是否打开新窗口
 

友情链接数据示例

Array (
    [0] => Array (
        [link_id] => 4 //字段 ID
        [link_name] => baigo //链接名称
        [link_url] => http://www.baigo.net //链接地址
        [link_blank] => 1 //是否打开新窗口
    )
)
