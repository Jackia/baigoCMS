图片、附件

在发布文章时，插入到文章的第一张图片或附件会被系统记录，在需要时，可以调用相关信息，图片、附件信息一般命名为 $this->tplData['articleRow']['attachRow']，在文章列表等处，也可能是多维数组的某一个键，如 $this->tplData['articleRows'][0]['attachRow']，详细结构如下。

 
键名 类型 说明 备注
attach_id int 附件 ID  
attach_name string 原始文件名  
attach_time int 上传时间戳  
attach_ext string 扩展名  
attach_size int 附件大小  
attach_url string 附件 URL  
 
top
 

缩略图

在上传图片时，系统会根据缩略图的设置自动生成缩略图，系统生成缩略图以后，需要在适当的地方给予显示，如：文章列表等处，一般缩略图信息包含在 attachRow 数组中。

以栏目主页的文章列表数组为例，缩略图位于 $this->tplData['articleRows'][0]['attachRow']，此数组包含了第一篇文章的所有图片信息，包括原始图片、多个缩略图等，如果我们要显示某一个尺寸的缩略图，我们需要得到调用键名，调用键名位于 附件管理 中的缩略图部分，如果我们要显示系统默认的 100x100 的缩略图，代码为 $this->tplData['articleRows'][0]['attachRow']['thumb_100_100_cut']。

 
数据示例
print($this->tplData['articleRows'][0]['attachRow'])
