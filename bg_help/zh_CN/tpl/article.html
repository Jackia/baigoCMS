文章显示

文件名：article_show.php

用于显示当前文章的详细信息。

键名 类型 说明 备注
$this->tplData['articleRow'] array 当前文章的详细信息 查看 $this->tplData['articleRow']。
$this->tplData['associateRows'] array 关联文章的列表 系统将一个或以上 TAG 相同的文章列出。此数组为多维数组，详细信息查看 $this->tplData['articleRow']。
 

$this->tplData['articleRow'] 数组

当前文章的详细信息

键名 类型 说明 备注
article_id int 文章 ID
article_title string 文章标题
article_content text 文章内容
article_cate_id int 隶属栏目 ID
article_mark_id int 隶属标记 ID
article_excerpt string 文章摘要
article_link string 跳转至 如填写了跳转地址，该文章将直接跳转至相应的地址，不会显示文章内容。
article_hits_day int 一天点击数
article_hits_week int 一周点击数
article_hits_month int 一月点击数
article_hits_year int 一年点击数
article_hits_all int 总点击数
article_time int 文章创建时间 指文章创建到数据库的时间。
article_time_show int 文章显示时间 指显示在文章中的时间。
article_is_time_pub int 是否定时上线 是否定时上线，值为 1 时文章定时上线。
article_time_pub int 文章上线时间 指文章上线时间。
article_is_time_hide int 是否定时下线 是否定时上线，值为 1 时文章定时下线。
article_time_hide int 文章下线时间 指文章下线时间。
urlRow array 当前文章 URL article_url 为文章 URL，page_ext 为文章扩展名。
tagRows array 当前文章关联的 TAG 所有与此文章关联的 TAG。详情请查看 TAG。
cateRow array 当前文章所属栏目的详细信息 详情请查看 栏目。
attachRow array 当前文章附件的详细信息 详情请查看 图片 / 缩略图。
article_status string 文章状态 pub 为发布，wait 为待审，hide 为隐藏。
article_customs array 自定义字段值 自定义字段的内容。用 echo $this->tplData['articleRow']['article_customs']['custom_自定义字段 ID'] 的方式便可显示自定义字段的内容。详情请查看通用资源 自定义字段，或后台管理 自定义字段。
article_source string 文章来源 常用于转载文章时，表明文章的出处，以示对原作者的尊重。
article_source_url string 文章来源 URL 常用于转载文章时，表明文章的出处，以示对原作者的尊重。
article_author string 文章作者 常用于转载文章时，表明文章的出处，以示对原作者的尊重。
rcode string 返回代码 显示当前文章的状态，详情请查看 返回代码。

文章计数

由于纯静态页面无法计数，因此在此提供计数接口，详情请查看 计数接口

此接口建议使用 script 或 iframe 方式调用，如 <script src="http://www.domain.com/bg_api/api.php?mod=article&act=hits&article_id=9" type="text/javascript"></script>
