调用

 

纯静态模式下

纯静态模式下，调用将会生成静态文件，调用方式可以在后台管理的查看调用中查询，推荐生成 html 文件，然后在需要显示调用的地方，用“服务器端嵌入 Server Side Include（SSI）”的方式来调用，如：<!--#include file="./call/9.html" -->。

调用模板位于 ./bg_tpl/call 目录下，如默认模板 ./bg_tpl/call/call_cate.php。注：模板文件名必须使用 英文 与 数字，不能使用中文、符号等。

默认模板结构说明
调用类型 默认模板 传递数据
文章列表、日排行、周排行、月排行、年排行、总排行 call_article.php
  • $this->tplData['callRow'] 调用 查看详情
  • $this->tplData['articleRows'] 文章列表 查看详情

  • $this->tplData['callRow'] 调用 查看详情

  • $this->tplData['callRow'] 调用 查看详情
  • $this->tplData['specRows'] 专题列表 查看详情

  • $this->tplData['callRow'] 调用 查看详情

当前栏目详细信息

键名 类型 说明 备注

其他模式下

在任何模板内，均可以用函数 fn_callDisplay(调用 ID) 的方式来调用，执行此函数后，会返回一个多维数组，可以通过遍历数组的方式来显示调用结果，返回数组会根据不同的调用类型有所不同，详情请查看 文章、栏目、TAG、专题、友情链接 等有关信息。关于调用以及调用 ID 请查看 调用管理。

调用附件

在任何模板内，均可以用函数 fn_callAttach(附件 ID) 的方式来调用，执行此函数后，会返回一个数组，可以通过遍历数组的方式来显示调用结果，详情请查看 附件 / 缩略图。

调用栏目

在任何模板内，均可以用函数 fn_callCate(栏目 ID) 的方式来调用，执行此函数后，会返回一个数组，可以通过遍历数组的方式来显示调用结果，详情请查看 栏目。
