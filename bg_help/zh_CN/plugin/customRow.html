自定义字段数组

名称 类型 描述
custom_id int 添加字段时，ID 为 0。
custom_name string 字段名
custom_type string 类型，可能的值为 str（输入框）、radio（单选）、select（下拉菜单）。
custom_status string 字段状态，可能的值为 enable（启用）、disable（禁用）。
custom_opt array 选项
custom_parent_id int 隶属于字段 ID
custom_cate_id int 隶属于栏目 ID
custom_format string 字段格式，可能的值为 text（文本）、date（日期）、datetime（日期时间）、int（整数）、digit（数字）、url（URL）、email（邮箱地址）。
custom_require bool 是否为必填字段
 
数据示例

Array (
    [custom_id] => 4
    [custom_name] => 尺寸
    [custom_type] => str
    [custom_opt] => Array ()
    [custom_status] => enable
    [custom_parent_id] => 0
    [custom_cate_id] => 13
    [custom_format] => int
    [custom_require] => 0
)
