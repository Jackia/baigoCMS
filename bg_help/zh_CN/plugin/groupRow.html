群组数组

名称 类型 描述
group_id int 添加群组时，ID 为 0。
group_name string 群组名
group_note string 备注
group_type string 群组类型，可能的值为 admin（管理员组）、user（用户组）。
group_status string 群组状态，可能的值为 enable（启用）、disable（禁用）。
group_allow array 管理权限
 
数据示例

Array (
    [group_id] => 1
    [group_name] => 超级管理组
    [group_note] => 拥有所有权限
    [group_allow] => Array (
        [article] => Array (
            [browse] => 1
            [add] => 1
            [edit] => 1
            [del] => 1
            [approve] => 1
            [tag] => 1
            [mark] => 1
            [custom] => 1
            [source] => 1
        )
        [spec] => Array (
            [browse] => 1
            [add] => 1
            [edit] => 1
            [del] => 1
        )
    )
    [group_type] => admin
    [group_status] => enable
    [rcode] => y040102
)
