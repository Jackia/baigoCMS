管理员信息数组

名称 类型 描述
admin_id int 用户 ID
admin_name string 用户名
admin_note string 备注
admin_nick string 昵称
admin_group_id int 隶属群组 ID
admin_type string 管理员类型，可能的值为 normal（普通管理员）、super（超级管理员）。
admin_status string 用户状态，可能的值为 enable（启用）、disable（禁用）。
admin_time int 注册时间，Unix 时间戳。
admin_time_login int 最后登录时间，Unix 时间戳。
admin_ip string 最后登录 IP 地址
admin_allow_cate array 栏目管理权限
admin_allow_profile array 个人信息修改权限
admin_prefer array 偏好设置
