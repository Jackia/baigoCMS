专题数组

名称 类型 描述
spec_id int 添加专题时，ID 为 0。
spec_name string 专题名称
spec_status string 专题状态，可能的值为 show（显示）、hide（隐藏）。
spec_content string 专题介绍
spec_attach_id int 附件 ID
 
数据示例

Array (
    [spec_id] => 9
    [spec_name] =>  文章管理专题
    [spec_status] => show
    [spec_content] =>
    [spec_attach_id] => 0
    [urlRow] => Array (
        [spec_path] => /wwwroot/cms/spec/9/
        [spec_pathShort] => /spec/9/
        [spec_url] => /cms/spec/9/
        [spec_urlMore] => /cms/spec/id-9/
        [page_attach] => page-
        [page_ext] => .html
    )
)
