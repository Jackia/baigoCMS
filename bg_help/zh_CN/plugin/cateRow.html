栏目数组

名称 类型 描述
cate_id int 添加栏目时，ID 为 0。
cate_name string 栏目名称
cate_alias string 栏目别名（用于 URL）
cate_type string 类型，可能的值为 normal（普通）、single（单页）、link（跳转）。
cate_status string 栏目状态，可能的值为 show（显示）、hide（隐藏）。
cate_tpl string 模板
cate_content string 栏目介绍（cate_type 为 single 时有效）
cate_link string 链接（cate_type 为 link 时有效）
cate_parent_id int 隶属于栏目 ID
cate_domain string URL 前缀
cate_perpage int 每页显示数
cate_trees array 栏目树
