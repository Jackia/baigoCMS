应用数组

名称 类型 描述
app_id int 添加应用时，ID 为 0。
app_name string 应用名
app_note string 备注
app_status string 应用状态，可能的值为 enable（启用）、disable（禁用）。
app_time int 注册时间，Unix 时间戳。
app_ip_allow string 允许调用的 IP 地址（白名单）
app_ip_bad string 禁止调用的 IP 地址（黑名单）
app_allow array 应用权限
 
数据示例

Array (
    [app_id] => 1
    [app_name] => tests
    [app_note] => test
    [app_status] => enable
    [app_time] => 1423880092
    [app_ip_allow] =>
    [app_ip_bad] =>
    [app_allow] => Array (
        [cate] => Array (
            [read] => 1
            [list] => 1
        )
        [article] => Array (
            [read] => 1
            [list] => 1
        )
        [spec] => Array (
            [list] => 1
        )
        [mark] => Array (
            [list] => 1
        )
    )
)
