链接数组

名称 类型 说明 备注
link_id int 链接 ID 添加链接时，ID 为 0。
link_name string 链接名称  
link_type string 链接类型 console 为后台，friend 为友情链接，auto 为自动链接。详情请查看后台管理 链接
link_url string 链接地址  
link_status string 链接状态 enable 为启用，disable 为禁用
link_cate_id int 隶属栏目 ID 仅在类型为 auto 时有效。
link_order int 排序  
link_blank bool 是否打开新窗口  
 
数据示例

Array (
    [link_id] => 0
    [link_name] => baigo Studio
    [link_type] => friend
    [link_url] => http://www.baigo.net
    [link_status] => enable
    [link_cate_id] => 3
    [link_order] => 0
    [link_blank] => 0
)
