数据库对象

$GLOBAL['obj_db'] 是系统中的一个全局对象，您可以在任何地方使用这个对象。该对象的类文件位于 ./bg_core/class/database.class.php，更为详细的使用可以参考该文件的源代码。

 
top
 

属性

 
属性 类型 说明
obj_mysqli object MySQLi 对象，详情请查看 PHP 官方网站。
obj_result object mysqli_result 对象，详情请查看 PHP 官方网站。
db_host string 数据库服务器地址
db_port string 数据库服务器端口
db_name string 数据库名称
db_user string 数据库用户名
db_pass string 数据库密码
db_charset string 数据库编码，一般为 utf8，请按照服务器提供商所提供的资料填写。
db_debug bool 是否开启数据库 debug 模式

方法

 
方法 说明
query 对数据库执行一次查询
create_table 创建一个数据表
create_index 创建一个索引
create_view 创建一个视图
copy_table 复制一个表格
alter_table 改变一个表格
insert 插入一条记录

delete 删除记录
count 统计记录

query($sql)

说明

对数据库执行一次查询。

参数

名称 类型 必需 描述
sql string true 钩子的名称，详情请查看 钩子列表。

返回

失败时返回 FALSE，成功执行 SELECT, SHOW, DESCRIBE 或 EXPLAIN 查询会返回一个 mysqli_result 对象，其他查询则返回 TRUE。

create_table($table, $data, $primary[, $comment = ''[, $engine = 'InnoDB']])

说明

创建一个数据表

参数

名称 类型 必需 描述
table string true 数据表的名称
data array true 数据表结构
primary string true 主键
comment string false 备注
engine string false 数据引擎，默认为 InnoDB。
 

返回结果

请查看 query。

 
data 结构示例

$articleCreat = array(
    //'字段名'    => '字段结构',
    'article_id'    => 'int NOT NULL AUTO_INCREMENT COMMENT "ID"',
    'article_title' => 'varchar(300) NOT NULL COMMENT "标题"',
    'article_link'  => 'varchar(900) NOT NULL COMMENT "链接"',
    'article_time'  => 'int NOT NULL COMMENT "记录时间"'
);

alter_table($table[, $data = false[, $rename = false]])

说明

改变一个表格

参数

名称 类型 必需 描述
table string true 数据表的名称
data array false 至少选择一个 表格结构
rename string 重命名
 

返回结果

请查看 query。

 
data 结构示例

$articleAlter = array(
    'article_upfile_id' => array( //'字段名'
        'CHANGE',//动作, 可选 ADD (添加)、DROP (丢弃)、CHANGE (变更)、ADD PRIMARY KEY (添加主键)、DROP PRIMARY KEY (丢弃主键)
        'int NOT NULL COMMENT "附件ID"', //字段结构
        'article_attach_id' //新字段名, 动作为 CHANGE 时必需
    ),
    'article_upfile_id' => array(
        'CHANGE',
        'int NOT NULL COMMENT "附件ID"',
        'article_attach_id'
    ),
    'article_cate_id' => array(
        'CHANGE',
        'smallint NOT NULL COMMENT "隶属栏目ID"',
        'article_cate_id'
    ),
    'article_mark_id' => array(
        'CHANGE',
        'smallint NOT NULL COMMENT "标记ID"',
        'article_mark_id'
    ),
    'article_time_day' => array(
        'ADD',
        'tinyint NOT NULL COMMENT "日点重置时间"',
    )
);

select($table[, $data = ''[, $where = ''[, $group = ''[, $order = ''[, $length = 0[, $start = 0[, $field = false]]]]]]])

    'article_time_pub',

);

$order = array(
    array(
        'article_top', //字段名

        'article_time_pub',
