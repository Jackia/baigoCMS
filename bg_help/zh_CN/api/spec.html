专题列表

接口说明

用于显示网站内所有的专题。

URL

http://www.domain.com/bg_api/api.php?mod=spec

HTTP 请求方式

GET

返回格式

JSON

请求参数
名称 类型 必须 具体描述
act string true 接口调用动作，值只能为 list。
app_id int true 应用的 APP ID，后台创建应用时生成的 ID。详情查看 查看应用。
app_key string true 应用的 APP KEY，后台创建应用时生成的 KEY。详情查看 查看应用。
key string false 搜索关键词
per_page int false 每页显示专题数
isBase64 string false 返回结果是否进行 Base64 编码。可选 true、false，如值为 false 返回结果为 json_encode 编码。
 
返回结果
名称 类型 说明 备注
specRows array 专题列表 详情请查看 专题显示返回结果。
pageRow array 分页参数 详情请查看 分页参数

专题显示

接口说明

用于显示当前专题的详细信息。

URL

http://www.domain.com/bg_api/api.php?mod=spec

HTTP 请求方式

GET

返回格式

JSON

请求参数
名称 类型 必须 具体描述
act string true 接口调用动作，值只能为 read。
app_id int true 应用的 APP ID，后台创建应用时生成的 ID。详情查看 查看应用。
app_key string true 应用的 APP KEY，后台创建应用时生成的 KEY。详情查看 查看应用。
spec_id int true 专题 ID
isBase64 string false 返回结果是否进行 Base64 编码。可选 true、false，如值为 false 返回结果为 json_encode 编码。
 
返回结果
名称 类型 说明 备注
spec_id int 专题 ID
spec_name string 专题名称
spec_content string 专题内容
spec_status string 专题状态 pub 为发布，hide 为隐藏。
rcode string 返回代码 显示当前专题的状态，详情请查看 返回代码。
 

返回结果示例

{
    "spec_id": "11", //专题 ID
    "spec_name": "专题名称", //专题名称
    "spec_status": "show", //状态
    "spec_content": "
专题内容
", //内容
    "urlRow": {
        "spec_url": "/cms/spec/id-11/", //URL
        "page_attach": "page-", //分页附加
        "page_ext": ".html"
    },
    "rcode": "y180102" //返回代码
}
