标记列表

接口说明

用于显示网站内所有的标记。

URL

http://www.domain.com/bg_api/api.php?m=mark&c=api

HTTP 请求方式

GET

返回格式

JSON

请求参数

名称 类型 必需 描述
a string true 接口调用动作，值只能为 list。
app_id int true 应用的 APP ID，后台创建应用时生成的 ID。详情查看 查看应用。
app_key string true 应用的 APP KEY，后台创建应用时生成的 KEY。详情查看 查看应用。
isBase64 string false 返回结果是否进行 Base64 编码。可选 true、false，如值为 false 返回结果为 json_encode 编码。
 

返回结果

名称 类型 说明 备注
mark_id int 标记 ID  
mark_name string 标记名称  
rcode string 返回代码 显示当前标记的状态，详情请查看 返回代码。
 
返回结果示例

[
    {
        "mark_id": "3", //标记 ID
        "mark_name": "推荐" //名称
    },
    {
        "mark_id": "4", //标记 ID
        "mark_name": "推荐到首页" //名称
    }
]
