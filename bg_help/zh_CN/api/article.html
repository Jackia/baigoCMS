文章列表

接口说明

用于显示文章列表。

URL

http://www.domain.com/bg_api/api.php?m=article&c=api

HTTP 请求方式

GET

返回格式

JSON

 

请求参数

名称 类型 必需 描述
a string true 接口调用动作，值只能为 list。
app_id int true 应用的 APP ID，后台创建应用时生成的 ID。详情查看 查看应用。
app_key string true 应用的 APP KEY，后台创建应用时生成的 KEY。详情查看 查看应用。
key string false 搜索关键词
year string false 年份
month string false 月份
cate_id int false 栏目 ID
mark_ids string false 标记 ID，多个 ID 请使用 | 分隔。
tag_ids string false
多个 ID 请使用 | 分隔。此三个参数为三选一，优先级依次为 tag_ids > spec_ids > customs。
关于 customs 详情请查看 搜索
spec_ids string false
customs string false
per_page int false 每页显示文章数
isBase64 string false 返回结果是否进行 Base64 编码。可选 true、false，如值为 false 返回结果为 json_encode 编码。
 

返回结果

键名 类型 说明 备注
articleRows array 文章列表 符合搜索条件的所有文章。详情请查看 文章显示返回结果。
pageRow array 分页参数 详情请查看 分页参数。

文章显示

接口说明

用于显示当前文章的详细信息。

URL

http://www.domain.com/bg_api/api.php?m=article&c=api

HTTP 请求方式

GET

返回格式

JSON

 

请求参数

名称 类型 必需 描述
a string true 接口调用动作，值只能为 list。
app_id int true 应用的 APP ID，后台创建应用时生成的 ID。详情查看 查看应用。
app_key string true 应用的 APP KEY，后台创建应用时生成的 KEY。详情查看 查看应用。
article_id int true 文章 ID
isBase64 string false 返回结果是否进行 Base64 编码。可选 true、false，如值为 false 返回结果为 json_encode 编码。
 

返回结果

名称 类型 说明 备注
article_id int 文章 ID  
article_title string 文章标题  
article_content text 文章内容  
article_cate_id int 隶属栏目 ID  
article_mark_id int 隶属标记 ID  
article_excerpt string 文章摘要  
article_link string 跳转至 如填写了跳转地址，该文章将直接跳转至相应的地址，不会显示文章内容。
article_hits_day int 一天点数  
article_hits_week int 一周点数  
article_hits_month int 一月点数  
article_hits_year int 一年点数  
article_hits_all int 总点数  
article_time int 文章创建时间 指文章创建到数据库的时间。
article_time_show int 文章显示时间 指显示在文章中的时间。
article_is_time_pub int 是否定时上线 是否定时上线，值为 1 时文章定时上线。
article_time_pub int 文章上线时间 指文章上线时间。
article_is_time_hide int 是否定时下线 是否定时上线，值为 1 时文章定时下线。
article_time_hide int 文章下线时间 指文章下线时间。
tagRows array 当前文章关联的 TAG 所有与此文章关联的 TAG。详情请查看 TAG。
cateRow array 当前文章所属栏目的详细信息 详情请查看 栏目。
attachRow array 当前文章附件的详细信息 详情请查看 附件。
article_status string 文章状态 pub 为发布，wait 为待审，hide 为隐藏。
article_customs array 自定义字段 自定义字段的内容。用 article_customs['custom_自定义字段 ID'] 的方式便可获取自定义字段的内容。详情请查看 自定义字段。
article_source string 文章来源 常用于转载文章时，表明文章的出处，以示对原作者的尊重。
article_source_url string 文章来源 URL 常用于转载文章时，表明文章的出处，以示对原作者的尊重。
article_author string 文章作者 常用于转载文章时，表明文章的出处，以示对原作者的尊重。
rcode string 返回代码 显示当前文章的状态，详情请查看 返回代码。

文章计数

接口说明

由于纯静态页面无法计数，因此在此提供计数接口，建议只用于纯静态模式。本接口只能提供简单的文章点数，如果需要详细的访问统计，请使用专业的网站统计系统。

URL

http://www.domain.com/bg_api/api.php?m=article&c=api

HTTP 请求方式

GET

返回格式

JSON

 

请求参数

名称 类型 必需 描述
a string true 接口调用动作，值只能为 hits。
article_id int true 文章 ID
 

返回结果

键名 类型 说明 备注
rcode string 返回代码 显示技术结果，详情请查看 返回代码。
