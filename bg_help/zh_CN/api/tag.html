TAG 显示

接口说明

用于显示当前 TAG 的详细信息。

URL

http://www.domain.com/bg_api/api.php?m=tag&c=api

HTTP 请求方式

GET

返回格式

JSON

请求参数

名称 类型 必需 描述
a string true 接口调用动作，值只能为 read。
app_id int true 应用的 APP ID，后台创建应用时生成的 ID。详情查看 查看应用。
app_key string true 应用的 APP KEY，后台创建应用时生成的 KEY。详情查看 查看应用。
tag_id int true TAG ID
isBase64 string false 返回结果是否进行 Base64 编码。可选 true、false，如值为 false 返回结果为 json_encode 编码。
 

返回结果

名称 类型 说明 备注
tag_id int TAG ID  
tag_name string TAG 名称  
tag_article_count string 文章计数 与当前 TAG 关联的文章计数。
tag_status string TAG 状态 pub 为发布，hide 为隐藏。
rcode string 返回代码 显示当前 TAG 的状态，详情请查看 返回代码。
 
返回结果示例

{
    "tag_id": "94", //TAG ID
    "tag_name": "asp", //TAG 名称
    "tag_status": "show", //状态
    "tag_article_count": "0", //关联文章数
    "urlRow": {
        "tag_url": "/cms/tag/tag-asp/", //URL
        "page_attach": "page-", //分页附加
        "page_ext": ".html"
    }
    "rcode": "y130102"
}
