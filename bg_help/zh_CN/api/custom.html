自定义字段列表

接口说明

用于显示系统内所有有效自定义字段列表。

URL

http://www.domain.com/bg_api/api.php?m=custom&c=api

HTTP 请求方式

GET

返回格式

JSON

请求参数

名称 类型 必需 描述
a string true 接口调用动作，值只能为 list。
app_id int true 应用的 APP ID，后台创建应用时生成的 ID。详情查看 查看应用。
app_key string true 应用的 APP KEY，后台创建应用时生成的 KEY。详情查看 查看应用。
isBase64 string false 返回结果是否进行 Base64 编码。可选 true、false，如值为 false 返回结果为 json_encode 编码。
 

返回结果

名称 类型 说明 备注
custom_id int 自定义字段 ID  
custom_name string 自定义字段名称  
rcode string 返回代码 显示当前 TAG 的状态，详情请查看 返回代码。
 
返回结果示例

[
    {
        "custom_id": "4", //字段 ID
        "custom_name": "尺寸", //名称
        "custom_parent_id": "0", //隶属字段 ID
        "custom_cate_id": "0", //隶属栏目 ID
        "custom_childs": { //子字段
            [
                {
                    "custom_id": "5",
                    "custom_name": "长",
                    "custom_parent_id": "4",
                    "custom_cate_id": "2"
                }
            ]
        }
    }
]
