SSO 设置

baigo CMS 的用户以及后台登录需要 baigo SSO 支持，baigo SSO 的部署方式，请查看 baigo SSO 官方网站。如果您的网站没有部署 baigo SSO，请点“SSO 自动部署“。

 
填写说明

API 接口 URL

baigo SSO API 接口的 URL

APP ID

baigo SSO 应用的 APP ID

APP KEY

baigo SSO 应用的 APP KEY

同步登录

如为开启状态，当用户在本站登录的时候，所有部署在 baigo SSO 下的网站将同步登录，当用户切换到这些网站时，无需再次登录。

 

点“保存“，成功后点“下一步“。
